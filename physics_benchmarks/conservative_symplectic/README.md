# Conservative / Hamiltonian benchmark (symplectic & position-only adapted)

The simple undamped pendulum (θ̈ = -(g/L) sin θ) is the native problem for:
- **Symplectic integrators** (semi_implicit_euler, leapfrog, ruth4): preserve phase-space structure, long-term energy stability
- **Second-order ODE specialists** (velocity_verlet, runge_kutta_nystrom, numerov): exploit θ̈ = a(t,θ) structure

Convergence and performance data for this benchmark are generated by the C++ integration test `SerialIntegrationSimplePendulumConvergesWithIncreasingResolutionAcrossIntegrators` and written to:
- `tests/artifacts/convergence/convergence_results.csv`
- `tests/artifacts/convergence/performance_by_order.png`
